# Assets Drop Zone

Put your files here using this structure:

- `assets/audio/` for audio files like `.mp3`, `.wav`, `.m4a`
- `assets/icons/` for app icons, UI icons, and SVG/PNG icon exports
- `assets/gifs/` for animated `.gif` files
- `assets/images/` for images like `.png`, `.jpg`, `.webp`

Suggested naming style:

- use lowercase
- use dashes instead of spaces
- example: `crowd-intro.mp3`, `hilal-badge.png`, `goal-loop.gif`

When you finish adding assets, tell me the filenames and I will wire them into the app.
